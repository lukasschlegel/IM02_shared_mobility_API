#AA Javascript Learnings/Schwierigkeiten
Wir haben die Programmiersprache JavaSript verwendet, um auf die vorgegebene API (https://github.com/SFOE/sharedmobility/blob/main/Sharedmobility.ch-API.md#get-find) von Shared Mobility anzuzapfen und die gewonnen Daten in einer interaktiven Karte anzuzeigen. Für diese interaktive Karte haben wir auf die Dienste von MapBox zurückgegriffen und mithilfe eines Gartis Accsess Token eine Basic Karte für unsere Webseite erstellt, die wir im Verlauf des Projekts immer weiter verfeinert und angepasst haben (Styling, Startpunkt, Suchfunktion, Geolocator). Durch die Integration der Daten der API in die Map konnten wir nun Scooter von verschiedenen Anbietern anzeigen lassen. Die Schwierigkeit bestand darin, die Scooter immer nur an diesem Ort anzeigen zu lassen, wo man im Suchfeld gesucht hat. Um die Info der Koordinaten der Suche in der Suchfunktion zu extrahieren und in das Javascript einzubauen war nicht einfach zu bewerkstelligen. Ein gutes Learning ist die richtige verwendung von "Dot Notations" und die richtige Integrierung von Objekten in "Template Strings", sowie die praktische Anwendung von Funktionen und Arrays.
